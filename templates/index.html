Click green circle to add card, red circle to remove card. Click and hold on a card image to zoom in on it.

To use deck in Tabletop Simulator, export JSON file to the following filepath:
WINDOWS: Documents > My games > Tabletop Sim > Saved > Saved Objects
MAC: User > Libary > Tabletop sim > Saved > Saved Objects
LINUX: ~/.local/share/Tabletop Simulator/Saves/Saved Object
